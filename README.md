# DaUtil --  云南数聚基础工具类
## 环境搭建
---
1. 安装gradle, 去[官网](https://gradle.org/gradle-download/)下载v3.x版本,并安装;
2. 在命令行中进入DaUtil目录, 执行以下命令编译打包并提交到本地Maven缓存:
```shell
cd DaUtil
gradle build publishToMavenLocal
```
